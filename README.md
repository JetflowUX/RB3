Pexels Integration
------------------

This project can optionally fetch curated images from the Pexels API at runtime. To enable this set your Pexels API key in a `.env` file using the `VITE_PEXELS_KEY` variable. See `.env.example` for the variable name.

If no key is provided the site falls back to Unsplash Source queries so images still load during development.
